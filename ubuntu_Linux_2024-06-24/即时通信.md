# 2.客户端：Thelounge

1.  安装npm

``yum install npm``

2. 安装yarn

``npm install --global yarn``

3. 安装 thelounge

`` yarn global add thelounge ``

4. 添加用户及密码

`` thlounge add 用户名``

![thelounge-user.png](http://123.56.103.61:9999/pic/home/bl/img/U1/thelounge-user.png)

5. 启动服务

`` thelounge start ``
